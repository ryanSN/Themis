# Themis
A Simple Discord chat bot using the Discord.js(https://github.com/hydrabolt/discord.js/) API library.

## Getting Started
currently this bot can only be self hosted.  


# Commands
#### Help:
 - `!ping`
 - `!uptime`

## Local Development / installation
####
- install [node.js]
- Run `npm install`
- Run `npm start`
